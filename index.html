📚 المرجع الشامل لكل وسوم HTML

كله بالعربي + أمثلة حية. احفظ الصفحة وذاكر أوفلاين

1. قسم الـ <head> - الحاجات اللي مش بتظهر

<meta charset="UTF-8">

بيعمل إيه؟ بيخلي العربي والإيموجي يظهروا صح 100%. لازم يبقى أول حاجة

<meta name="viewport" content="width=device-width, initial-scale=1.0">

بيعمل إيه؟ بيظبط الموقع عالموبايل. من غيره الموقع يطلع مصغر أوي

كل وسوم HTML - شرح عربي

📚 كل وسوم HTML5 - شرح عربي مبسط

احفظ الصفحة دي وذاكر منها أوفلاين. كل وسم تحته شرحه بالعربي

1. الهيكل الأساسي

<!DOCTYPE html> : بيقول للمتصفح دي صفحة HTML5. لازم تبقى أول سطر

<html> : بداية ونهاية الصفحة كلها. كل حاجة جواها

<head> : فيها الإعدادات والعنوان. مش بتظهر في الصفحة

<meta charset="UTF-8"> : عشان العربي يظهر صح. ملهوش قفلة

<title> : اسم الصفحة اللي بيظهر فوق في التاب

<body> : كل المحتوى اللي الناس بتشوفه

<!DOCTYPE html>
<html>
<head><title>عنواني</title></head>
<body>المحتوى</body>
</html>

2. العناوين والفقرات

<h1> ل <h6> : عناوين من كبير لصغير. h1 أكبر واحد

<p> : فقرة/براجراف. بينزل سطر جديد

<br> : سطر جديد بس. ملهوش قفلة

<hr> : خط فاصل بين الأقسام. ملهوش قفلة

<h1>عنوان كبير</h1>
<p>دي فقرة</p>
<br>
<hr>

3. تنسيق النص

<b> : عريض Bold

<strong> : عريض ومهم أكتر للـ SEO

<i> : مائل Italic

<em> : مائل ومؤكد عليه

<u> : تحته خط

<mark> : تظليل أصفر زي الماركر

<small> : خط صغير

<del> : مشطوب عليه

<ins> : تحته خط للإضافة

<b>عريض</b> <mark>متظلل</mark> <del>ملغي</del>

4. الروابط والصور

<a href="الرابط"> : لينك. href هو المكان اللي هيروحله

<img src="الصورة.jpg" alt="وصف"> : صورة. src مكانها، alt لو الصورة مظهرتش. ملهاش قفلة

ملحوظة: alt مهم عشان جوجل ولو النت ضعيف

<a href="https://google.com">روح جوجل</a>
<img src="cat.jpg" alt="صورة قطة">

5. القوائم

<ul> : قائمة نقط

<ol> : قائمة أرقام

<li> : عنصر جوه القائمة

<ul>
<li>تفاح</li>
<li>موز</li>
</ul>

6. الجداول

<table> : الجدول كله

<tr> : صف Table Row

<td> : خانة عادية Table Data

<th> : عنوان الخانة، بيبقى عريض ومتوسط

<table border="1">
<tr><th>الاسم</th><th>العمر</th></tr>
<tr><td>أحمد</td><td>20</td></tr>
</table>

7. الفورم - النماذج

<form> : استمارة بيانات

<input type="text"> : مربع كتابة. ملهوش قفلة

<input type="password"> : كلمة سر بتستخبي

<textarea> : مربع كتابة كبير

<select> : قائمة منسدلة

<option> : اختيار جوه القائمة

<button> : زرار

<label> : اسم الحقل اللي جنب المربع

<form>
<label>اسمك:</label>
<input type="text">
<button>إرسال</button>
</form>

8. التقسيم والحاويات

<div> : صندوق كبير بيقسم الصفحة - Block

<span> : صندوق صغير جوه السطر - Inline

<header> : رأس الصفحة

<nav> : القائمة/التنقل

<main> : المحتوى الأساسي

<section> : قسم من الصفحة

<article> : مقال مستقل

<footer> : تذييل الصفحة

9. وسوم HTML5 الجديدة

<video> : فيديو

<audio> : صوت

<canvas> : لوحة رسم

<progress> : شريط تحميل

جرب التفاعل بـ JavaScript

عدد الوسوم اللي شوفتها: 0

دي أهم 80% من الوسوم. الباقي استخدامات متقدمة 💪

<meta name="description" content="الوصف">

بيعمل إيه؟ الوصف اللي بيظهر تحت اسم موقعك في جوجل. اكتب 160 حرف

<meta name="keywords" content="كلمات">

بيعمل إيه؟ كلمات مفتاحية لجوجل زمان. دلوقتي جوجل مبقاش يبص عليها

<meta http-equiv="refresh" content="10">

بيعمل إيه؟ بيعمل تحديث للصفحة كل 10 ثواني تلقائي

ملحوظة: كل وسوم meta ملهاش قفلة </meta>

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="موقعي لتعليم البرمجة">

2. وسوم <link> - الربط بالملفات الخارجية

<link rel="stylesheet" href="style.css">

بيعمل إيه؟ بيربط ملف CSS خارجي عشان الألوان والشكل. rel = نوع العلاقة، href = مكان الملف

<link rel="icon" href="logo.png">

بيعمل إيه؟ بيحط الأيقونة الصغيرة اللي جنب اسم التاب فوق. اسمها Favicon

<link rel="preconnect" href="رابط">

بيعمل إيه؟ بيسرع تحميل الخطوط والصور من موقع تاني

ملحوظة: link ملهاش قفلة برضو

<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.png">

3. الصور <img>

<img src="الصورة.jpg" alt="وصف">

بيعمل إيه؟ بيحط صورة في الصفحة

src = مكان الصورة Source

alt = الوصف البديل. بيظهر لو الصورة مبطلعش، ومهم لجوجل

width و height = العرض والطول بالبكسل

ملحوظة: img ملهاش قفلة

<img src="cat.jpg" alt="صورة قطة" width="300" height="200">

4. الفيديو <video>

<video controls width="400">

بيعمل إيه؟ بيشغل فيديو في الصفحة

controls = بيظهر أزرار التشغيل والإيقاف

autoplay = يشتغل لوحده أول ما الصفحة تفتح

loop = يعيد الفيديو كل ما يخلص

muted = يشغل من غير صوت

<video controls width="400">
  <source src="فيديو.mp4" type="video/mp4">
</video>

5. الصوت <audio>

<audio controls>

بيعمل إيه؟ بيشغل ملف صوتي MP3

نفس الصفات بتاعة الفيديو: controls, autoplay, loop

<audio controls>
  <source src="صوت.mp3" type="audio/mpeg">
</audio>

6. الإطار <iframe>

<iframe src="رابط">

بيعمل إيه؟ بيجيب صفحة موقع تاني جوه صفحتك. زي فيديو يوتيوب

width و height = المقاس

title = وصف للإطار

<iframe src="https://youtube.com" width="500" height="300"></iframe>
